# WanderStay 🏨

A MERN stack travel booking platform inspired by Airbnb.

## Features
- Property listings with photos
- Search and filters
- User authentication (JWT)
- Booking management
- Responsive UI

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Auth: JWT

## Setup
npm install
npm start
